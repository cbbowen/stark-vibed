# Stability

## Drawing Files

* **Alpha** (current): No guarantees whatsoever. Files may not load at all in future versions. If you care about a piece, save it as a image file.
* **Beta**: Files will continue to load in future versions, but portions may be lost or changed.
* **Release**: Files will continue to load and be perceptually similar. Note that this does not guarantee that will reproduce pixel-perfectly, even with the same version.
